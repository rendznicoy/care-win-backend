# Election Tally Board (Backend)

![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54) ![Next JS](https://img.shields.io/badge/Next-black?style=for-the-badge&logo=next.js&logoColor=white) ![Status](https://img.shields.io/badge/Status-Discontinued-inactive?style=for-the-badge)

> **The server-side infrastructure and API services for the Election Tally Board.**

This repository houses the backend logic, API endpoints, and database management systems that power the Election Tally Board application. It handles vote processing, data persistence, and serves the necessary API routes for the frontend visualization.

**Project Status:**
This project is currently **Discontinued** and is no longer being actively maintained or updated. The code is archived here for portfolio and reference purposes.

**Frontend Repository:**
[Tally Board Frontend](https://github.com/rendznicoy/care-win)

---

## Key Features

* **API Management:** Robust endpoints for handling election data requests and submission.
* **Data Processing:** Server-side logic to calculate vote tallies and rankings accurately.
* **Database Integration:** Secure storage and retrieval of candidate and voting records.

---

## Tech Stack

* **Language:** Python
* **Framework:** Next.js
* **Database:** Supabase
* **Package Manager:** npm / pip
